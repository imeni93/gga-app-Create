📖 使用说明
• 适配机型：国兴 7070 台达包覆纱机

• 原料规格：70D 锦纶 + 70dtex 氨纶

• 核心常数：K1 = 1345.996，K2 = 3741.07

• 齿轮范围：30 ~ 80 偶数齿

• 正向计算：输入工艺参数 → 推荐 2 套最优齿轮 + 产量

• 反向计算：输入齿轮参数 → 反推工艺 + 产量

• 方案收藏：点击 ⭐ 收藏方案，可添加备注

• 长按回填：收藏列表中长按 0.5 秒 → 确认回填

• 回填后自动运算：回填参数后自动计算并出结果

• 收藏分组：按原料规格自动分组，可折叠/展开

• 深色模式：点击右上角「🌓 主题」切换

• 快捷操作：输入框内按回车键即可快速计算

• 数据缓存：参数自动保存，下次打开自动恢复
